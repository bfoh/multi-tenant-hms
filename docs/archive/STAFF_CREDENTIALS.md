## Features Available

### All Staff Members
- View dashboard with occupancy and revenue stats
- Manage reservations (view, create, edit, cancel)
- Create on-site bookings
- View calendar
- Access guest information
- Generate invoices
- End of day reports

### Admin Only
- Admin panel access
- Set room prices
- Manage additional services
- Configure local tax
- Manage meals

## Troubleshooting

If login fails:
1. Ensure you're entering the credentials exactly as shown above
2. Check your internet connection
3. Clear browser cache and try again
4. Contact support if the issue persists

## Security Notes

- Passwords are hashed and stored securely
- All staff actions are logged and associated with user accounts
- Session expires after period of inactivity
- Staff access is verified on each page load
